# PyScript Bad Apple

![Terminal Bad Apple](./resource/terminal.mov)
![Web Bad Apple](./resource/web.mov)

## Run on Terminal

### use `uv`

```sh
uv venv
uv add -r requirements.txt 
uv run main.py
```

### use `pip`

```
pip install -r requirements.txt
python3 main.py
```


### Run on web

```sh
python3 -m http.server
```
